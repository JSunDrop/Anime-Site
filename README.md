# OtakuHub — Grande Pro V3.2 (Static, GH Pages–safe)
This is a minimal, GH‑Pages‑friendly build (with a professional landing) to help you verify your deployment.
All assets are referenced **relatively**, and scripts are **defer**-loaded with a crash banner if something fails.

## Deploy
- Put these files at your Pages root (or `/docs` if that’s what GitHub Pages uses).
```
index.html
app.js
data.js
bg.js
style.css
/assets/placeholder.svg
```
- Visit `https://<username>.github.io/<repo>/#/landing`
